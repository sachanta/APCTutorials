Testing with Command Line Options

Appvance PerformanceCloud provides a CommandLine ScriptRunner to execute shell scripts as functional tests, load and performance tests, and production monitors.

Make calls to shell scripts using the Command Line Test Type in the Scenario Builder. The first argument must be /c on Windows machines to identify a shell window command. The second argument is the shell script command. Appvance PerformanceCloud logs a success or failure based on the return value - 0 for success and 1 for failed operation.
